## 1. Projeto

### 1.1 Contexto
Este é um projeto para desenvolvimento de um jogo destinado a Comfy Jam: Autumn edição de 2025. Tem como objetivo desenvoler jogos com temática agradável e tranquila, utilizando elementos estéticos do outono.

### 1.2 Restrição
Como em todos os anos dessa Jam, existe uma retrição: O jogo deverá começar com uma cutscene. Dado isso, decidimos utilizar ela como oportunidade para apresentar a história e contextualizar o jogador.

## 2. Jogo

### 2.1 Conceito
Considerando a premissa da Game Jam, optamos por desenvolver um jogo sem mecânicas de combate, pois não parecia se encaixar bem com a proposta. Ao invés disso escolhemos apostar em um jogo focado na interação com o mapa e NPCs com elementos de puzzle e stealth.

### 2.2 Gráficos
Os gráficos foram definidos como em pixel art, no formato de 32x32, fortemente inspirados pelo estílo de jogos como Deltarune, Earthbound 3 e The legend of Zelda: A Link to the past.

### 2.3 Gameplay
O jogo tem como principal mecânica o stealth. O jogador terá como objetivo retornar itens para determinados personagens espalhados pelo mapa, mas sem que seja visto. Ele terá de descobrir onde estão os itens utilizando os diálogos dos NPCs, da mesma forma como descobrirá para quem esses mesmos itens deverão ser retornados. Caso o jogador seja visto, ela perderá **uma** de **três** vidas, recomeçando o level. Caso suas vidas se esgotem o jogo recomeçará, porém, ao entregar um 1 item ao seu dono, uma vida será recuperada.

### 2.4 História
O jogador personifica Baltazar, um fantasma amigável que adora os costumes humanos e estar em sua companhia. Ele "assombra" uma pequena cidade, onde, todos os anos no outono, realiza uma comemoração, na qual os cidadãos se unem e celebram. Nesse ano, no entanto as festividades correm o risco de serem canceladas, agora, é dever de Baltazar descobrir o motivo e ajudar no que puder, mas sem que seja visto assustando os moradores e piorando ainda mais a situação.
